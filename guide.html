Course Guide: Introduction to HTML and Web Development

Subject: Computer Applications Technology (CAT)
Grade: 10
Curriculum: CAPS-aligned
Course Duration: 3 Weeks
Format: Theory and practical sessions

Course Overview

This course introduces Grade 10 learners to the foundational principles of web design and development using HTML (HyperText Markup Language).
Rooted in the South African CAPS curriculum for Computer Applications Technology, this course empowers learners with the essential digital literacy and technical skills needed for building simple, functional web pages.
Through engaging lessons and hands-on activities, learners will explore the structure of HTML documents, various HTML tags and elements, and basic web development techniques.



Course Aims and Objectives

By the end of this course, learners will be able to:

  • Understand what HTML is and its role in web development.
  • Write and interpret basic HTML code.
  • Use various HTML tags and elements to structure content.
  • Create and display simple web pages using text, images, and links.
  • Demonstrate foundational knowledge of web development practices.


Module Breakdown



Module 1: Introduction to HTML

Objective: To introduce learners to the concept of HTML and its application in building web pages.
Topics Covered:
What is HTML?
The purpose of HTML in web development
Structure of a basic HTML document [< ! DOCTYPE html >, "< html>,"" < head>, < body>]
The importance of web browsers and text editors (e.g., Notepad, VS Code)
Activities:
Identifying HTML in real-world websites
Writing your first "Hello, world!" HTML page
Assessment:
Short quizzes

Objective: To introduce learners to the concept of HTML and its application in building web pages.

Basic tags

Module 3: Web Development

Objective: To combine knowledge from previous modules to build simple, styled websites.
Topics Covered:
Creating multi-page websites using internal linking
Saving, testing HTML pages

Build a small website project
Assessment:

Assessment Methods
