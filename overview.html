Course Overview: HTML and Web Development for Grade 10 CAT


This short course introduces Grade 10 Computer Applications Technology (CAT) learners to the basics of HTML and web development, following the CAPS curriculum. Learners will explore how websites are structured and created using HTML code. The course covers the structure of a web page, the use of various tags and elements, and the creation of simple multi-page websites. By the end of the course, learners will be able to write and test basic HTML code, understand web page components, and build a functional website using text, images, and links.

Course Modules

  • 1. Introduction to HTML – Understanding the role of HTML and creating basic web pages
  • 2. Tags and Elements – Using HTML tags, attributes, and formatting content
  • 3. Web Development – Building and styling a simple website project

Requirements

To complete this course, you’ll need access to a computer, a text editor like Notepad++, and a web browser. Prior coding experience is not required.
